.. title:: Welcome

**Welcome to the Sprite documentation!**

Sprite is a Python-based implementation of the Functional-Logic Programming
language Curry.  It seamlessly combines Functional, Logic, and Imperative
programming in a single environment, so **you** get to decide which is best for
your application.
