\subsection{Detailed Description}
A reference-counting pointer for {\bf Node}{\rm (p.\,\pageref{structsprite_1_1Node})} objects. 

This is based on boost::intrusive\_\-ptr, so the reference count is stored within the {\bf Node}{\rm (p.\,\pageref{structsprite_1_1Node})} object itself (to maximize performance). All dereferencing operations skip over FWD nodes, which are nodes that simply forward their reference to a new target. Those may occur when a node with multiple external references is rewritten.
