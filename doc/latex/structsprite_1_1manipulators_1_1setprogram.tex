\subsection{Detailed Description}
I/O manipulator to set the program. 

The program is required to look up label names. A pointer to the program is contextual information stored with an output stream until cleared and is requried to stream out {\bf Node}{\rm (p.\,\pageref{structsprite_1_1Node})} objects.
